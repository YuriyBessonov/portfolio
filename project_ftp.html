FTP клиент для Linux

Клиентское приложение для взаимодействия с сервером по протоколу FTP. Работа клиента осуществляется в пассивном режиме. Программа позволяет осуществлять обмен файлами с сервером и поддерживает основные операции управления файлами и каталогами на сервере.

При разработке использовались: C, ncurses.
