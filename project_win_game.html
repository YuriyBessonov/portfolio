Сетевая многопользовательская игра

Многопользовательская игра для Windows (шутер), представленная клиентским и серверным приложением. Задача игроков - набрать как можно больше очков за попадание в противников. Игроки идентифицируются по имени, отображаемому в списке результатов вместе с указанием числа набранных очков.

При разработке использовались: C++, SFML, WinSock.
