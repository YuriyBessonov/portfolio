Игровое приложение для Android (Жуки)

Приложение представляет собой игру, в которой игрок должен давить генерируемых в случайных местах жуков, передвигающихся по экрану, получая за это очки. Исключение - божья коровка, раздавив которую игрок получает штраф.

При разработке использовались: Java, Android SDK.
